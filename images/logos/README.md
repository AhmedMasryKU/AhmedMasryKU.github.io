# Logo Images

Place logo images in this folder. The images should be:

- **Google Scholar**: `google-scholar.png` (recommended size: 16x16px or larger, will be scaled to 1em height)
- **HuggingFace**: `huggingface.png` (recommended size: 16x16px or larger, will be scaled to 1em height)

Supported formats: PNG, JPG, SVG

If a logo is missing, the text fallback will be displayed.
