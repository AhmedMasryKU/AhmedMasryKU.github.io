# Company Logos

Place company logo images in this folder. The images should be:

- **ServiceNow Research**: `servicenow.png` (recommended size: 80x80px or larger, will be scaled)
- **Arteria AI**: `arteria.png` (recommended size: 80x80px or larger, will be scaled)

Supported formats: PNG, JPG, SVG

If a logo is missing, a fallback with company initials will be displayed.
